## Development Instructions

1. Clone: `git clone https://github.com/reednel/pages.git`
2. Checkout the dev branch: `git checkout develop`
3. Use Node 22: `nvm use 22`
4. Install Node Modules: `npm i`
5. Build: `npm run dev`
